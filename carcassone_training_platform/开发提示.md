# 🧩 meta_features 示例结构

这是一个当前游戏中 `meta_features` 的实际示例，用于记录所有连通的 Feature 特征（如城市、道路、田地）及其所属 Meeple。

---

## 数据结构格式：

```python
meta_features: Dict[str, Dict[int, FeatureGroup]]
meta_features = {
    "CityFeature": {
        3: FeatureGroup(
            tiles={
                (0, 0): [2, 3],             # tile (0, 0) 上第 2、3 边属于此城市
                (0, -1): [0, 1, 6],         # tile (0, -1) 上第 0、1、6 边属于此城市
                (1, -1): [7]               # tile (1, -1) 上第 7 边属于此城市
            },
            meeples=[
                Meeple(id=0, owner="player1", type="normal", feature_id=3, active=True),
                Meeple(id=2, owner="player0", type="builder", feature_id=3, active=True)
            ]
        )
    },
    "RoadFeature": {
        5: FeatureGroup(
            tiles={
                (1, 0): [0, 3],            # tile (1, 0) 上第 0、3 边属于道路
                (2, 0): [6]               # tile (2, 0) 上第 6 边属于道路
            },
            meeples=[]
        )
    },
    "FieldFeature": {}
}
